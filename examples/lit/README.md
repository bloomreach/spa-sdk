# Lit CSR SPA

Example Lit SPA using the Bloomreach [Lit SDK](https://www.npmjs.com/package/@bloomreach/lit-sdk). The app is built
with [Vite](https://vitejs.dev/) and [Lit](https://lit.dev/) web components.

## Install and run

This example is part of a monorepo and other packages will have to be built in order for this example to run. Read more
in the [root README](../../README.md#development) on how to install, build and run the examples.

## Available scripts

In the project directory, you can run:

### `pnpm run dev`

Runs the app in development mode. Open <http://localhost:3010> to view
it in the browser.

The page will reload if you make edits.

### `pnpm run build`

Builds the app for production to the `dist` folder. It bundles the
application and optimizes the build for the best performance.

The build is minified and the filenames include hashes. Your app is ready
to be deployed!

### `pnpm run start`

Runs the app in production mode. This will start a static file server to serve
the app from the `dist` folder. This requires the app to have been built first
by running `pnpm run build`.

### `pnpm run preview`

Preview the production build locally using Vite's built-in preview server.
